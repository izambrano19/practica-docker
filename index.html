SERVIDOR WEB APACHE



¿QUÉ ES UN SERVIDOR WEB?


Un servidor web es un programa que procesa y devuelve información cuando un usuario realiza una solicitud.
Es decir, aloja sitios web en uno o varios servidores y muestra el contenido (sitios web) del servidor a tráves de un navegador.



¿QUÉ ES UN SERVIDOR WEB APACHE?


Es un servidor web HHTP de código abierto. Está desarrollado y mantenido por una comunidad de usuarios en torno a la Apache Software Fundation.
Es el servidor web más utilizado en todo el mundo por la estabilidad y seguridad que esta otorga.



PRINCIPALES CARACTERÍSTICAS


  • Es gratuito y de código abierto.
  • Fácil de instalar y configurar.
  • Altamente extensible y adaptable con módulos.
  • Funciones incorporadas para la autentificación y validación de usuarios.
  • Soporte para lenguajes como Perl, PHP y Python.


5 VENTAJAS


  • Soporte: Es uno de los servidores más usados desde hace más de 2 décadas, por lo que detrás hay una comunidad muy amplia.
  • Multiplataforma: Funciona tanto en Microsoft Windows como en Unix/Linux y Mac.
  • Características: Contiene un gran número de módulos oficiales y de terceros que le permiten ampliar la funcionalidad del servidor.
  • Simplicidad: Es uno de los servidores más fáciles de instalar y configurar.
  • Seguridad: Lleva incorporadas funciones para hacer seguro el software con módulos de autorización y autentificación y funciones de control, así como soporte para la encriptación con certificados SSL/TLS.


2 DESVENTAJAS


  • El rendimiento puede verse afectado en páginas de alto tráfico, especialmente después de 10.000 conexiones.
  • Si los módulos no se utilizan correctamente, se pueden crear agujeros de seguridad.
